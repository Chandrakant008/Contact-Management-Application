# Contact Management Application

A **web-based Contact Management Application** built using **Python and Django** that allows users to manage their contacts efficiently with full CRUD (Create, Read, Update, Delete) functionality. This application features a clean and responsive user interface using **Bootstrap**.

---

## **Project Overview**
This application allows users to store, view, update, and delete contact information. Each contact contains the following details:

- First Name  
- Last Name  
- Address  
- Email  
- Phone Number  

The app ensures **data validation**, **no duplicates for email or phone numbers**, and provides a **user-friendly interface** for easy navigation.

---

## **Features**
- **Create Contact:** Add a new contact with all required details.  
- **Read / Display Contacts:** View a list of all saved contacts in a table.  
- **Update Contact:** Edit existing contact details.  
- **Delete Contact:** Remove a contact with confirmation.  
- **Data Validation:** Validates email format and ensures no duplicate email or phone numbers.  
- **Responsive Design:** Clean interface using Bootstrap that works on all devices.

---

## **Technology Stack**
- **Backend:** Python, Django  
- **Database:** SQLite (default for Django projects)  
- **Frontend:** HTML, CSS, Bootstrap  
- **Development Environment:** VS Code or any Python IDE  
- **Version Control:** Git & GitHub

## **Installation & Setup**
1. **Clone the repository**
   
git clone https://github.com/Chandrakant008/Contact-Management-Application.git
cd Contact-Management-Application

2. Create a virtual environment
python -m venv venv

3.Activate the virtual environment
venv\Scripts\activate

4.Install dependencies
pip install -r requirements.txt

5.Run migrations
python manage.py makemigrations
python manage.py migrate

5.Run the development server
python manage.py runserver
